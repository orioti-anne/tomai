# 1차 프로젝트

## 서비스 개요
- 스마트팜 재배환경과 생육 데이터를 활용하여 생산량을 예측하고 출하 적기를 추천하는 서비스 개발 프로젝트입니다.
- 기상 정보와 가격 데이터를 함께 분석하여 농가의 재배 및 출하 의사결정 지원을 목표로 합니다.

## 개발 목표
- 재배정보, 생육 데이터, 생산 데이터를 활용하여 생산량 예측 모델 구축
- 재배환경 및 생육 데이터를 기반으로 생육속도 예측 모델 구축
- 기상 정보 기반 생육 속도와 가격 예측을 종합 분석하여 출하 적기 추천 서비스 구축
