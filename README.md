# 스마트팜 출하 적기 추천 AI 플랫폼 

## 프로젝트 개요
 스마트팜의 재배 환경 및 생육 데이터를 기반으로 미래 생산량을 예측하고 출하 시점을 추천하는 의사결정 지원 시스템입니다.
 기상 정보와 시장 가격 데이터를 함께 분석하여 농가의 수익 극대화와 농산물 수급 안정화를 목표로 합니다.

## 개발 목표
- **생산량 예측 모델**
    - 재배 환경, 생육 상태, 과거 생산 데이터를 기반으로 수확량 예측 모델 구축
- **생육 속도 분석**
    - 주요 환경 변수(온도, 습도, CO₂, 누적일사량 등)에 따른 작물의 생육 단계 및 성장 속도 추정
- **출하 적기 추천**
    - 외부 데이터(기상청, KAMIS)를 API 연동하여 최적의 출하 시점 제안
