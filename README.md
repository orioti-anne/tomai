# 스마트팜 출하 적기 추천 AI 솔루션 - tomAI

## 프로젝트 개요
 스마트팜의 재배 환경 및 생육 데이터를 기반으로 미래 생산량을 예측하고, 
 기상 정보와 시장 가격 데이터를 함께 분석하여 최적의 출하 시점을 추천하는 AI 솔루션입니다.

 주요 목표는 다음과 같습니다.
- 농가의 예상 수확량 및 매출 예측
- 재배 환경에 따른 생육 속도 예측
- 시장 가격과 기상 변화를 반영한 출하 적기 추천
- 농가의 수익 극대화 및 농산물 수급 안정화

## 주요 기능
- **생산량 예측**
    - 재배 환경, 생육 상태, 재배 이력 데이터를 기반으로 예상 수확량 산출
    - 면적당 생산량 및 총 생산량 예측
    - 예상 수확일 및 남은 기간 계산
    - 예상 매출 자동 계산
- **생육 상태 분석**
    - 최근 생육 정보를 기반으로 개체별 성장 분석
    - 초장, 생장길이, 엽수, 엽폭, 개화수, 착과수 등 주요 지표 추적
    - 최근 변화량과 평균 성장 속도 계산
- **환경 데이터 분석**
    - 온도, 습도, CO₂, 토양온도, 누적일사량 등 센서 데이터 수집
    - 일별·누적 환경 정보 계산
    - 최근 7일·14일 평균 환경 데이터 자동 생성
    - 센서 이상치 및 누락 데이터 자동 보정
- **출하 적기 추천**
    - 기상청 기상 API와 KAMIS 가격 데이터 연동
    - 향후 기상 변화와 시장 가격 흐름 분석
    - 가격 예측 기반의 출하 시나리오 제공

## 개발 목표
- **생산량 예측 모델 구축**
    - 재배 환경·생육 상태·과거 생산 데이터를 기반으로 수확량 예측
- **생육 속도 분석 모델 구축**
    - 주요 환경 변수에 따른 작물의 생육 단계 및 성장 속도 추정
- **출하 적기 추천 모델 구축**
    - 외부 데이터(API)를 활용하여 최적의 출하 시점 및 예상 수익 제안
- **환경 제어 추천**
    - 현재 생육 상태에 따라 적정 온도·습도·CO₂ 조건 및 고온주의 기준 제안

## 시스템 구성
 1. 데이터 입력
 2. raw data 적재 → 데이터 정제·보정 → Summary data 생성 
 3. AI 모델 학습 및 예측 수행 
 4. 생산량·가격·출하 적기 결과 제공

## 데이터 구조
- **입력 데이터**
    - 재배 기본 정보
      - 작기, 정식일, 면적, 재식밀도, 품종 등
    - 환경 데이터
      - 외기온, 내기온, 습도, CO₂, 토양온도, 일사량 등
    - 생육 데이터
      - 초장, 생장길이, 엽수, 엽폭, 엽장, 화방높이, 개화수, 착과수 등
    - 생산 데이터
      - 출하일, 판매량, 판매금액 등
- **주요 파생 변수**
  - 적산 온도·적산 일사량
  - 최근 7일·14일 평균 환경
  - 생육 변화량
  - 면적당 생산량
  - 예상 남은 수확 기간
  - 예상 총 매출
